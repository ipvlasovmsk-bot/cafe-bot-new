# Инструкция по установке QR-кода для оплаты

## 📋 Настройка

### 1. Создайте QR-код для оплаты

**Вариант А: Через приложение банка**
1. Откройте приложение вашего банка
2. Найдите раздел "Платежи" → "QR-код для оплаты"
3. Сгенерируйте QR-код для вашей карты
4. Сохраните изображение QR-кода

**Вариант Б: Через онлайн-генератор**
1. Перейдите на сайт генератора QR-кодов (например, qr-code-generator.com)
2. Введите данные для оплаты (номер карты, получатель)
3. Скачайте сгенерированный QR-код

**Вариант В: Через СБП (Система Быстрых Платежей)**
1. В приложении банка создайте QR-код для СБП
2. Укажите сумму (или оставьте поле пустым для произвольной суммы)
3. Сохраните изображение

### 2. Разместите файл с QR-кодом

Поместите изображение QR-кода в корневую папку проекта с именем `payment_qr.jpg`

Или укажите свой путь в `.env`:
```env
PAYMENT_QR_PATH=path/to/your/qr_code.png
```

**Поддерживаемые форматы:**
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

### 3. Обновите конфигурацию

В файле `.env` укажите путь к QR-коду:
```env
PAYMENT_QR_PATH=payment_qr.jpg
```

### 4. Перезапустите бота

```bash
python main.py
```

## ✅ Как это работает

Когда пользователь нажимает "Оплатить" в корзине:
1. Бот проверяет наличие файла с QR-кодом
2. Если файл существует — отправляет фото QR-кода с текстом оплаты
3. Если файл не найден — отправляет только текст с номером карты

## 📸 Пример

Пользователь увидит:
```
💳 Оплата заказа на 1500₽

Переведите сумму на карту:
2200 1111 2222 3333

📱 Или отсканируйте QR-код

[ФОТО QR-КОДА]

✅ Подтвердить оплату
❌ Отмена
```

## 🔧 troubleshooting

**QR-код не отображается:**
- Проверьте путь в `.env`
- Убедитесь что файл существует
- Проверьте права доступа к файлу
- Посмотрите логи бота (`bot.log`)

**Неверный формат:**
- Конвертируйте изображение в JPG или PNG
- Убедитесь что файл не повреждён
- Проверьте размер файла (рекомендуется до 5MB)
